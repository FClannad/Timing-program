# 拼豆店计时管理系统

现代化 Web 界面的座位计时管理系统，简洁优雅。

## 使用方法

1. 进入 `dist` 文件夹
2. 双击 `拼豆计时系统.exe`
3. 浏览器自动打开系统界面

## 功能特点

✅ 49个座位实时管理  
✅ 套餐自定义（时长+价格）  
✅ 超时自动提醒  
✅ 历史记录查询  
✅ 今日统计报表  
✅ 收费状态管理  
✅ 现代化界面设计  

## 技术栈

- 后端：Flask (Python)
- 前端：HTML5 + CSS3 + JavaScript
- 数据库：SQLite3
- 打包：PyInstaller

## 数据存储

```
C:\Users\你的用户名\PinDouTimer\拼豆店数据.db
```

## 系统要求

- Windows 7 / 10 / 11
- 无需安装 Python
- 无需额外依赖
