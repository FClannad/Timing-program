# 拼豆店计时管理系统 - 使用说明

## 快速开始

### 双击运行
```
进入 dist 文件夹
双击 拼豆计时系统.exe
浏览器会自动打开系统
```

---

## 功能特点

### 座位管理
- 一号大厅：1-31号
- 二号大厅：32-49号
- 实时显示座位状态
- 超时座位自动提醒

### 套餐管理
- 添加、编辑、删除套餐
- 自定义时长（分钟）
- 自定义价格（元）

### 历史记录
- 查看所有订单
- 顾客信息详情
- 收费状态管理

### 今日统计
- 已收/未收金额
- 订单数量统计
- 套餐销售明细

---

## 数据存储

数据保存在：
```
C:\Users\你的用户名\PinDouTimer\拼豆店数据.db
```

定期备份此文件即可保护数据。

---

## 系统要求

- Windows 7/10/11
- 无需安装Python
- 无需其他依赖
